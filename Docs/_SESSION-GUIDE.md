# Session guide (FixIt)

**Start here each coding session** — then run the **`/start`** Cursor command for the full checklist.

1. Read **`AGENTS.md`** (repo root).
2. Read **`.cursor/rules/*.mdc`** (project invariants).
3. Skim **`docs/project.md`** + **`docs/specs.md`**.
4. Read **last entries** in **`docs/progress.md`**, **`docs/todos.md`**, **`docs/chatSummaries.md`**.
5. If touching auth, Google, mobile native, or API contracts: **`docs/_CONNECTIONS.md`** + **`docs/debugs.md`**.
6. Deep product questions: **`docs/fixit_implementation_guide (1).md`** (by section).

End session: **`/end`**. Mid-session note: **`/checkpoint`**.
